## 🛠️ Setup

1. **Clone the repository**:
   `bash
git clone <https://Aliexe-code/PS-UT-Typescript.git>
cd PS-UT-Typescript
bun install
     `
2. **Running files**

```bash
   bun run 'path-to-file'
```

3.**Running tests**

```bash
    bun test
    bun test tests/'path-to-file'
```
